📐 Interpolation de Lagrange

Construire un polynôme passant par un ensemble de points donnés

⏳ Chargement de Python...

Principe théorique

L'interpolation de Lagrange consiste à trouver un polynôme unique de degré n-1 passant exactement par n points donnés.


Formule :

  • P(x) = Σ yᵢ · Lᵢ(x)
  • Lᵢ(x) = Π (x - xⱼ) / (xᵢ - xⱼ) pour j ≠ i

Algorithme :

  • Pour chaque point i, calculer le polynôme de base Li(x)
  • Multiplier par la valeur correspondante yi
  • Sommer tous les termes pour obtenir P(x)
🐍 lagrange.py 
import numpy as np

def lagrange(a, b, x):
    """
    a : liste des points x (noeuds)
    b : liste des valeurs f(x)
    x : point où on veut interpoler
    """
    n = len(a)
    result = 0.0

    for i in range(n):
        numerator   = 1.0
        denominator = 1.0
        for j in range(n):
            if j != i:
                numerator   *= (x - a[j])
                denominator *= (a[i] - a[j])
        result += b[i] * (numerator / denominator)

    return round(result, 8)
