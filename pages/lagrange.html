📐 Interpolation de Lagrange

Construire un polynôme passant par un ensemble de points donnés

⏳ Chargement de Python...

Principe théorique

L'interpolation de Lagrange permet de trouver l'unique polynôme de degré n-1 passant exactement par n points donnés.

  • Pour chaque point i, calculer le polynôme de base Li(x)
  • Multiplier par la valeur yi correspondante
  • Sommer tous les termes pour obtenir P(x)

Video explanation — Lagrange Interpolation

🐍 lagrange.py 
def lagrange(a, b, x):
    n = len(a)
    result = 0.0
    for i in range(n):
        num, den = 1.0, 1.0
        for j in range(n):
            if j != i:
                num *= (x - a[j])
                den *= (a[i] - a[j])
        result += b[i] * (num / den)
    return round(result, 8)
