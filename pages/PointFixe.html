🔄 Méthode du Point Fixe

Trouver x tel que g(x) = x par itérations successives

⏳ Chargement de Python...

Principe théorique

La méthode du point fixe transforme l'équation f(x) = 0 en x = g(x) et itère à partir d'une estimation initiale x0.


  • Choisir une estimation initiale x0
  • Itérer xn+1 = g(xn)
  • S'arrêter quand |xn+1 - xn| < ε

Video explanation — Fixed Point Method

🐍 point_fixe.py 
import math
import numpy as np

def point_fixe(g, x0, tol=1e-8, max_iter=10000):
    x = x0
    for n in range(max_iter):
        x_new = g(x)
        erreur = abs(x_new - x)
        if erreur < tol:
            print(f"Convergence en {n+1} itérations : x* = {x_new:.8f}")
            return x_new
        x = x_new
    print("Limite d'itérations atteinte.")
    return x
