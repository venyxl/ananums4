〰️ Méthode de Simpson

Intégration numérique par arcs de parabole

⏳ Chargement de Python...

Principe théorique

La méthode de Simpson approche l'intégrale en subdivisant l'intervalle en 2m sous-intervalles et en approximant la fonction par des arcs de parabole.


Algorithme :

  • h = (b - a) / (2m)
  • I ≈ (h/3) × [f(a) + f(b) + 4·f(x1) + 2·f(x2) + ...]
  • Coefficients : 1, 4, 2, 4, 2, ..., 4, 1
🐍 simpson.py 
import numpy as np

def simpson(f, a, b, m):
    h = (b - a) / (2 * m)
    s = 0
    steps = []

    for i in range(1, 2 * m):
        xi = a + i * h
        coeff = 2 if i % 2 == 0 else 4
        s += coeff * f(xi)
        steps.append({
            'i':      i,
            'xi':     round(xi, 6),
            'f(xi)':  round(f(xi), 6),
            'coeff':  coeff
        })

    I = (h / 3) * (f(a) + f(b) + s)
    return round(I, 8), steps
