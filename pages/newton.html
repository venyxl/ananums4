🔺 Interpolation de Newton

Interpolation par différences divisées

⏳ Chargement de Python...

Principe théorique

L'interpolation de Newton utilise les différences divisées pour construire un polynôme passant par n points.


Algorithme :

  • Construire la table des différences divisées
  • Évaluer le polynôme au point demandé
  • Résultat : P(x) = f[x0] + f[x0,x1](x-x0) + ...
🐍 newton.py 
import numpy as np

def newton(x, y, val):
    """
    x   : liste des noeuds
    y   : liste des valeurs f(x)
    val : point où on veut interpoler
    """
    n = len(x)
    m = np.zeros((n, n))
    m[:, 0] = y

    for i in range(1, n):
        for j in range(i, n):
            m[j, i] = (m[j, i-1] - m[j-1, i-1]) / (x[j] - x[j-i])

    result  = m[0, 0]
    product = 1.0

    for i in range(1, n):
        product *= (val - x[i-1])
        result  += m[i, i] * product

    return round(float(result), 8), m.tolist()
