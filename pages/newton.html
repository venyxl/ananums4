Interpolation par différences divisées

Principe théorique

L'interpolation de Newton utilise les différences divisées pour construire un polynôme passant par n points. Cette forme est plus efficace pour ajouter de nouveaux points.

Algorithme :

  • Construire le tableau des différences divisées
  • Évaluer le polynôme au point demandé
  • Résultat : P(x) = f[x0] + f[x0,x1](x-x0) + ...

🐍 newton.py 
import numpy as np

def newton(x, y, val):
    n = len(x)
    m = np.zeros((n, n))
    m[:, 0] = y
    for i in range(1, n):
        for j in range(i, n):
            m[j,i] = (m[j,i-1] - m[j-1,i-1]) / (x[j] - x[j-i])
    result, product = m[0,0], 1.0
    for i in range(1, n):
        product *= (val - x[i-1])
        result  += m[i,i] * product
    return round(float(result), 8)
