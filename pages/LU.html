🧩 Décomposition LU

Factorisation de A en L (inférieure) et U (supérieure) pour résoudre Ax = b

⏳ Chargement de Python...

Principe théorique

La décomposition LU factorise une matrice A sous la forme A = L × U, ce qui simplifie grandement la résolution de Ax = b en deux systèmes triangulaires.


  • Factorisation : Trouver L et U telles que A = LU
  • Descente : Résoudre Ly = b pour y
  • Remontée : Résoudre Ux = y pour x
  • Avantage : Réutiliser L,U pour plusieurs membres droits B

🐍 decomp_lu.py 
import numpy as np

def Decomp_LU(A, B):
    n = A.shape[0]
    L = np.zeros((n,n))
    for i in range(n-1):
        E = np.eye(n)
        for j in range(i+1, n):
            w = A[j,i] / A[i,i]
            E[j,i] = -w
        A = E @ A
        L = L + E
    L = -L
    for i in range(n): L[i,i] = 1
    U = A
    # Descente : Ly = B
    Y = np.zeros(n)
    Y[0] = B[0]
    for i in range(1, n):
        Y[i] = B[i] - sum(L[i,j]*Y[j] for j in range(i))
    # Remontée : Ux = Y
    X = np.zeros(n)
    for i in range(n-1, -1, -1):
        X[i] = (Y[i] - sum(U[i,j]*X[j] for j in range(i+1,n))) / U[i,i]
    return L, U, Y, X
