🔺 Élimination de Gauss

Résolution de systèmes linéaires par échelonnement et substitution arrière

⏳ Chargement de Python...

Principe théorique

L'élimination de Gauss transforme la matrice augmentée [A|b] en forme triangulaire supérieure, puis résout par substitution arrière.


  • Phase 1 — Échelonnement : Pour chaque pivot k, éliminer la colonne k sous la diagonale
  • Multiplicateur : w = A[i,k] / A[k,k]
  • Phase 2 — Remontée : Résoudre de bas en haut x[i] = (b[i] − Σ A[i,j]·x[j]) / A[i,i]

🐍 gauss.py 
import numpy as np

def Gauss(A, b):
    M = np.column_stack((A.astype(float), b.astype(float)))
    n = A.shape[0]
    for k in range(n-1):
        for i in range(k+1, n):
            w = M[i,k] / M[k,k]
            for j in range(k, n+1):
                M[i,j] = M[i,j] - w * M[k,j]
    X = np.zeros(n)
    for i in range(n-1, -1, -1):
        s = sum(M[i,j]*X[j] for j in range(i+1,n))
        X[i] = (M[i,n] - s) / M[i,i]
    return M, X
