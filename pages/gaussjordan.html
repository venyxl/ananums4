✨ Gauss-Jordan

Échelonnement réduit — la solution apparaît directement dans la dernière colonne

Principe théorique

Gauss-Jordan étend l'élimination de Gauss en continuant l'échelonnement au-dessus de chaque pivot, produisant une matrice identité à gauche. La solution X se lit directement dans la dernière colonne.


  • Normaliser chaque ligne pivot : M[i,:] = M[i,:] / M[i,i]
  • Éliminer dans toutes les autres lignes (au-dessus et en-dessous)
  • Résultat : [I | X] — pas besoin de substitution arrière !
🔍 Différence avec Gauss : Gauss élimine seulement en-dessous du pivot → substitution arrière nécessaire. 🐍 gauss_jordan.py 
import numpy as np

def Gauss_J(A, B):
    M = np.hstack((A, B.reshape(-1, 1))).astype(float)
    n = M.shape[0]
    for i in range(n):
        M[i,:] = M[i,:] / M[i,i]          # Normalisation
        for j in range(n):
            if i != j:
                M[j,:] = M[j,:] - M[j,i] * M[i,:]  # Élimination
    X = M[:,n]   # Solution directe !
    return X
